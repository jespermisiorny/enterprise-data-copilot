# Data Sources

## Kostnadsställe
Kostnadsställe används för att gruppera kostnader per organisatorisk enhet.

Exempel:
- Produktion
- Underhåll
- IT
- HR

Kostnadsställe används i både budget- och uppföljningsrapporter.

---

## Tidsdimension
Tidsdimensionen används för att analysera data per:
- Vecka
- Månad
- Kvartal
- År

Veckonummer följer ISO-standard.
